
#Full Stack Blog
            
## Description
A lightweight Node.js blog app   with user auth, CRUD for posts and categories, and static front-end assets.
## Installation
Use the login button at the top of the page, create an account. You will be taken to a user section where you can create blog posts. 
## Usage
Use the login button at the top of the page, create an account. You will be taken to a user section where you can create blog posts.
